# DualCam 📷

App iOS double caméra — avant + arrière simultanément.  
Photo et vidéo fusionnées en un seul fichier (PiP).

## Fonctionnalités
- 📸 Capture photo double (avant + arrière fusionnées, PiP draggable)
- 🎥 Enregistrement vidéo double (flux fusionnés, export galerie)
- ⚡ Flash toggle
- 🔍 Zoom pinch-to-zoom sur caméra arrière
- ⏱ Timer d'enregistrement
- 🇫🇷 Interface en français

## Prérequis
- iPhone avec **multi-camera** support (iPhone 11 ou plus récent)
- iOS 16.0 minimum

## Compiler avec Codemagic (depuis Windows)

1. Push ce repo sur GitHub
2. Va sur [codemagic.io](https://codemagic.io) → connecte ton GitHub
3. Crée un nouveau projet → sélectionne ce repo
4. Le fichier `codemagic.yaml` est déjà configuré
5. Lance le build → télécharge le `.ipa`
6. Installe sur iPhone via **Sideloadly** (Windows)

## Installer via Sideloadly
1. Télécharge [Sideloadly](https://sideloadly.io)
2. Connecte ton iPhone en USB
3. Glisse le `.ipa` dans Sideloadly
4. Entre ton Apple ID (gratuit, pas besoin de compte développeur payant)
5. L'app est installée — re-signe tous les 7 jours (limitation Apple ID gratuit)
